# Prof. Dr. S. Balaji Science Academy

Welcome to the official website of Prof. Dr. S. Balaji Science Academy — a dedicated coaching platform for Class 12 Board Exams (CBSE and TN State Board).

## About the Academy

With 25+ years of teaching experience, a PhD in Physical Chemistry from the University of Madras, and postdoctoral work at IIT Madras, Prof. Dr. S. Balaji brings unparalleled expertise to board exam preparation. Unlike mainstream coaching centers focused on entrance exams, we are built exclusively for board exam success.

## What This Repository Contains

This repository hosts the complete website for Prof. Dr. S. Balaji Science Academy, built with clean, minimal design principles and optimized for all devices — from mobile phones to desktops.

The site includes:
- **Main Website** — Information about courses, teaching methods, faculty, and how to enroll
- **CBSE Digest** — A dedicated page for our free YouTube lecture series covering Class 10 Science

## Features

- Fully responsive design (mobile-first approach)
- Smooth scroll animations and parallax effects
- Clean, professional, minimal aesthetic
- Fast-loading static pages built with vanilla HTML, CSS, and JavaScript

3. Commit and push changes to the `main` branch
4. Your changes will be live within seconds
